# REST Client

A Postman-like REST client built with Next.js and MikroORM.

## Features

- Make HTTP requests (GET, POST, PUT, DELETE, PATCH)
- View request history with pagination
- Save requests/responses to PostgreSQL database
- Clean, responsive UI
